# wechat-mass-sender

<div align="center">

**Windows 微信 PC 版自动群发工具** —— 深色专业 GUI + 命令行双模式

支持联系人和群聊、个性化消息、防风控延时、失败重试

</div>

---

## ✨ 特性

- 🎨 **深色专业 GUI** —— tkinter 自绘控件，卡片化布局，扁平现代风
- 👥 **联系人 + 群聊** —— 同一份名单文件同时支持，CSV 格式
- 🪄 **个性化内容** —— 消息模板中用 `{name}` 占位，发送时自动替换
- ⏱ **防风控延时** —— 每条之间随机间隔（默认 5–12 秒）
- 🔁 **失败重试** —— 每条最多自动重试 2 次
- 🪟 **可靠窗口激活** —— 用 `AttachThreadInput` + Alt 键技巧绕过 Windows 前台锁
- ⛔ **紧急停止** —— 发送中把鼠标移到屏幕左上角立即中断
- 📝 **完整日志** —— 同时输出到 GUI 面板和日志文件
- 🖥 **命令行版** —— 不喜欢 GUI 也能用，行为完全一致

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/<your-username>/wechat-mass-sender.git
cd wechat-mass-sender
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 准备收件人

编辑 `recipients.csv`：

```csv
name,type,message
文件传输助手,contact,这是一条测试消息，发送给 {name}
张三,contact,你好 {name}，提醒你明天上午 10 点参加项目评审
工作群,group,【通知】各位 {name} 成员，本周五前请提交周报
```

| 字段 | 必填 | 说明 |
|---|---|---|
| `name` | ✅ | 微信中显示的名称（联系人昵称或群名），**必须完全一致** |
| `type` | ✅ | `contact` 联系人 / `group` 群聊 |
| `message` | ✅ | 消息内容，可用 `{name}` 占位符 |

### 4. 启动

**GUI 模式**（推荐）：
```bash
python gui.py
```

**命令行模式**：
```bash
python wechat_mass_sender.py
```

### 5. 登录微信 + 保持窗口可见

> ⚠️ 微信窗口**必须保持前台可见**（不能最小化），脚本才能正常操作。

### 6. 点击「🚀 开始群发」→ 二次确认 → 自动发送

## ⚙️ 配置说明

GUI 中所有配置项可调：

| 配置项 | 默认 | 说明 |
|---|---|---|
| 最小延时 | 5 秒 | 每条消息发送后等待的最短时间 |
| 最大延时 | 12 秒 | 每条消息发送后等待的最长时间（实际为区间内随机） |
| 重试次数 | 2 | 单条发送失败时的自动重试次数 |
| 点击 X 比例 | 0.5 | 输入框水平位置（相对微信窗口，0~1） |
| 点击 Y 比例 | 0.85 | 输入框垂直位置（相对微信窗口，0~1，靠近底部） |
| 搜索等待 | 1.5 秒 | Ctrl+F 搜索后等待结果出现的时间 |

> 💡 如果点击位置不准，在 GUI 中微调 X/Y 比例后再次测试。

## 🧠 技术原理

微信 PC 客户端没有公开的自动化 API。本工具采用 **GUI 自动化** 方案：

1. **窗口激活**：`pywin32` 找到微信窗口 → `AttachThreadInput` + `Alt` 键技巧绕过 Windows 前台锁 → 用 `GetForegroundWindow()` 验证激活成功
2. **搜索联系人**：`Ctrl+F` 打开微信搜索 → 剪贴板粘贴名称 → `Enter` 进入聊天
3. **点击输入框**：用配置的 X/Y 比例定位到窗口底部输入区
4. **粘贴消息**：剪贴板写入内容 → `Ctrl+V` 粘贴 → `Enter` 发送
5. **每步验证**：每一步操作前都检查微信是否在前台，丢失则重新激活

## ❓ 常见问题

**Q: 提示「无法把微信窗口激活到前台」？**
A: 三种常见原因：
  1. 微信窗口被其他全屏应用遮挡
  2. 微信窗口已最小化
  3. 系统前台锁限制（罕见）

  解决方法：手动点一下微信窗口使其前置，再重新点发送。

**Q: 发送成功但对方没收到？**
A: 检查名称是否与微信显示完全一致（包含特殊符号、emoji 等）。

**Q: 微信版本兼容性？**
A: 支持微信 PC 版 3.x / 4.x。不依赖微信内部控件，UI 变化不影响自动化。

**Q: 会被封号吗？**
A: 本工具本质上是手动操作的自动化脚本，行为与人工操作一致。
  - 建议每条之间 5 秒以上延时
  - 单次群发不超过 50 人
  - 同一账号每天不超过 200 人
  - 避免发送违规内容

**Q: 支持图片/文件吗？**
A: 当前仅支持文本消息。图片/文件支持正在计划中。
